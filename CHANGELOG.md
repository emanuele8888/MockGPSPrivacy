## [1.0.0] - 2026-03-16

### Added
- Precise GPS simulation — set your location anywhere in the world via interactive map.
- **Kinematic Jitter Engine**: realistic GPS drift simulation using Hooke's Law, inertia, and damping physics.
- **Stealth Mode (Privacy Check)**: scans installed apps for potential log-readers or mock-detection tools.
- **Profile Management**: save, load, and delete favorite locations for quick switching.
- **Background Foreground Service**: keeps the mock location active even when the app is closed.
- Modern UI built with Jetpack Compose and Material 3.
- Support for Android API 26+ (Android 8.0 Oreo and above).
